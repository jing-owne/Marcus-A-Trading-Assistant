# Marcus A股交易助手使用指南

## 🎯 系统概述

Marcus A股交易助手是一个智能股票查询和市场分析系统，基于Marcus（15年华尔街经验的日内交易策略师）的分析模型，为您提供：

1. **股票查询**：输入股票名称或代码，获取实时涨幅、交易量、买入评级和建议
2. **市场观点**：输入"市场观点"，获取每日动量报告（Daily Momentum Report）

## 📋 核心功能

### 股票查询功能
- **实时股票分析**：输入股票名称或代码，获取实时数据
- **分析内容**：
  - 当前价格
  - 价格变化百分比
  - 交易量
  - 盘前成交量倍数
  - 技术信号
  - 催化剂事件
  - Marcus买入评级
  - 胜率概率
  - 买入理由和具体建议

- **支持的股票**：
  - A股：宁德时代(300750)、比亚迪(002594)、贵州茅台(600519)、东方财富(300059)、药明康德(603259)、迈瑞医疗(300760)、爱尔眼科(300015)、中芯国际(688981)、隆基绿能(601012)、长江电力(600900)
  - 港股/美股：腾讯控股(00700)、阿里巴巴(BABA)、美团点评(03690)、京东集团(JD)、百度(BIDU)、拼多多(PDD)、网易(NTES)、小米集团(01810)

### 市场观点功能
- **每日动量报告**：输入"市场观点"，获取Marcus的市场分析和5%观察名单
- **报告内容**：
  - Marcus的市场立场（激进买入/保守买入/持币观望）
  - 市场状况分析（VIX指数、股指期货、成交量变化）
  - 5%观察名单（5只胜率概率最高的股票）
  - 技术面和基本面分析
  - Marcus的忠告

## 🚀 使用方法

### 命令行启动
```bash
python Marcus_Final.py
```

### 参数模式
```bash

python Marcus_Final.py 宁德时代

# 获取市场观点
python Marcus_Final.py 市场观点
```

## 🛠️ 技术架构

### 主要模块
1. **Marcus_Final.py** - 主交互程序
2. **Marcus_Trading_Agent.md** - Marcus的人设配置和交易策略

### Marcus分析模型
- **技术面权重（60%）**：成交量异常、技术突破形态、关键价位接近度、RSI信号
- **基本面权重（40%）**：财报催化剂强度、行业催化剂强度、时间窗口临近度
- **综合胜率计算公式**：胜率 = (技术面得分 × 60%) + (基本面得分 × 40%)

### Marcus市场立场判断标准
- **激进买入**：上证指数上涨>1%，成交量放大>20%
- **保守买入**：市场震荡，上证指数±0.5%以内，成交量稳定
- **持币观望**：市场下跌，上证指数跌幅>1%，成交量下降
