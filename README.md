# Marcus A股交易助手 v3.0

> 基于15年华尔街+A股实战经验的日内动量策略系统 · 真实数据接入 · 全自动定时推送

---

## 🎯 系统概述

Marcus A股交易助手 v3.0 是一套**准实战级**A股自动化分析系统，通过 AkShare 接入真实市场数据，结合 MACD/KDJ 双金叉、连续净流入、量价突破等技术指标，每日自动筛选胜率 ≥70% 的动量标的，并在工作日关键时间节点自动推送报告。

---

## 🏗️ 系统架构

```
Marcus_Engine.py       # 核心数据引擎（AkShare数据接入 + 筛选逻辑 + 技术指标）
Marcus_Report.py       # 报告生成器（4个推送模块 + 命令行入口）
Marcus_Final.py        # 交互式查询模式（股票单查 + 市场观点）
README.md              # 本文档
```

## ✅ 核心筛选逻辑（硬约束）

| 筛选项 | 规则 |
|--------|------|
| 市场范围 | 仅沪深/创业板（600/000/001/002/003/300开头） |
| 排除科创板 | 剔除所有 688 开头股票 |
| 排除新股 | 上市未满30个交易日不纳入 |
| 流动性 | 日成交额 ≥2000万 且 换手率 ≥1% |
| 当日过滤 | 排除当日涨停（涨幅 ≥9.5%）|
| 昨日涨幅 | ≤5%（排除已高位标的）|
| 活跃度 | 20日内有涨停（题材仍在）|
| 总涨幅 | 20日内总涨幅 ≤20%（排除追高风险）|
| 技术信号 | MACD 金叉（DIF上穿DEA）+ KDJ 金叉（K上穿D）双确认 |
| 资金信号 | 连续3天净流入 |
| 胜率门槛 | ≥70% 方可入选观察名单 |

## 📈 市场立场判断依据

综合以下5个维度：
- **沪深300指数** 涨跌幅 + 趋势
- **iVIX（中国波指）** 恐慌/贪婪情绪
- **股指期货** 升贴水方向
- **全市场成交量** vs 5日均量比
- **富时A50 + 恒生科技** 外盘情绪

输出三档立场：
- 🟢 **激进买入**（Aggressive Buy）
- 🟡 **保守买入**（Conservative Buy）
- 🔴 **持币观望**（Hold / Cash）

## 🚀 快速使用

### 环境要求
```bash
Python 3.8+
pip install akshare pandas numpy
```

### 手动运行

```bash
# 盘中动量报告（15只名单）
python -X utf8 Marcus_Report.py intraday

# 收盘总结（板块流向 + 次日建议）
python -X utf8 Marcus_Report.py closing

# 开盘风险提醒
python -X utf8 Marcus_Report.py alert

# 交互式股票查询
python Marcus_Final.py
python Marcus_Final.py 宁德时代
python Marcus_Final.py 300750
```

### 自动化（通过 WorkBuddy 调度）

系统已配置5个工作日自动化任务，无需手动触发。每个时间节点自动拉取实时数据并生成报告。
