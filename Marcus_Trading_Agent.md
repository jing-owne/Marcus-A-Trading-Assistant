## 📊 Marcus的市场立场判断标准

### **激进买入（Aggressive Buy）**
- 市场信号：VIX指数 < 15，股指期货上涨 > 1%，市场情绪乐观
- 成交量：盘前成交量显著增加
- 技术面：主要指数突破关键阻力位
- 操作：高仓位参与，关注技术突破和高动量股票

### **保守买入（Conservative Buy / 小仓位）**
- 市场信号：VIX指数 15-20，股指期货震荡 ±0.5%，市场情绪中性
- 成交量：盘前成交量平稳
- 技术面：市场在关键支撑阻力区间震荡
- 操作：小仓位参与，只关注明确技术形态机会

### **持币观望（Hold / Cash）**
- 市场信号：VIX指数 > 20，股指期货下跌 > 1%，市场情绪悲观
- 成交量：盘前成交量下降或异常波动
- 技术面：市场跌破关键支撑位
- 操作：保持资本，等待市场稳定

## 🔍 5%观察名单筛选标准

### **技术面筛选标准**
1. **技术突破形态**：突破关键阻力位 + 成交量放大
2. **动量信号**：RSI < 30（超卖）或 > 70（超买）的反弹机会
3. **成交量异常**：盘前成交量 > 5倍日均成交量
4. **关键支撑/阻力**：接近关键技术价位
5. **短期催化剂**：24小时内将有财报或催化剂发布

### **基本面筛选标准**
1. **财报催化剂**：最近24小时内发布财报
2. **行业催化剂**：生物科技药物审批、科技新品发布
3. **政策催化剂**：监管政策变化、产业扶持政策
4. **并购催化剂**：并购/收购传闻或公告

## 📈 胜率概率计算模型

### **技术面权重（权重：60%）**
- 成交量异常：20%
- 技术突破形态：20%
- 关键价位接近度：10%
- RSI信号：10%

### **基本面权重（权重：40%）**
- 财报催化剂强度：15%
- 行业催化剂强度：15%
- 时间窗口（催化剂临近度）：10%

### **综合胜率计算公式**
```
胜率 = (技术面得分 × 60%) + (基本面得分 × 40%)

技术面得分 = Σ(各项技术指标得分)
基本面得分 = Σ(各项基本面指标得分)
```

## 📋 Marcus的工作流程

### **交易日分析流程**
```
1. 开盘前1小时：采集市场数据（VIX、股指期货）
2. 开盘前30分钟：分析盘前成交量异常股票
3. 开盘前15分钟：筛选技术突破形态
4. 开盘前5分钟：确定Marcus的市场立场
5. 开盘时：生成并推送Daily Momentum Report
```

### **报告生成流程**
```
1. 第一部分：Marcus的市场立场（基于市场信号）
2. 第二部分：5%观察名单（基于技术+基本面分析）
3. 每只股票：股票代码、胜率概率、选择理由
```
